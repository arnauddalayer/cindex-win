Using the Find panel

Use the Find panel (Edit menu) to search among records for words or phrases, or to find records that meet more complex search criteria.

To find records that contain a word or phrase, type the phrase beside the Not check box, then click Find. Cindex will highlight the first record that contains the specified text. Click Find Again to continue the search. You can open any highlighted record and work on it, then continue the search.

Once you have set up and found the first occurrence of a target entry, you need not leave the Find panel open while searching for subsequent occurrences. You can choose Find Again from the Edit menu.

To find all records that contain the word or phrase, click Find All. Cindex will form a “group” of the records that contain the text, and will display the group in the main window, as though it contained the only records in the index.

  • To find records that do not contain the specified text, check Not.
  • To find text set in particular styles or fonts click Attributes.
  • To search only among particular levels of heading, choose from the popup menu under Not.
    If you choose a particular heading level, and do not specify search text, Cindex will find records in which the specified heading is empty.
    If you choose Locator, Cindex normally treats the text as a locator specification, rather than plain text (the item Evaluate References is automatically checked). For more information see Searching for Locators.
  • To make Cindex accept only matches to whole words, check Whole Word.
  • To make Cindex accept only case-sensitive matches, check Case.
  • To make Cindex treat the search text as a pattern to be matched, construct the pattern in the text box and check Pattern.
  • To search among a subset of displayed records specify the Scope of the search.
  • To confine the search to records that have one or more special attributes check the relevant attributes in the box Only Among.
  • To limit the search to records created or modified between certain dates, enter the range of dates. (To find records changed on a particular day, enter that day'sdate at From, and enter the next day's date at To.) Cindex will warn you if it cannot understand a date you provide.
  • To confine the search to records made or modified by a particular user, enter the user's identifying initials at UserID.
