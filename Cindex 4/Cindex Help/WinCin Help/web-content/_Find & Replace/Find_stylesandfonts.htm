Finding text styles and fonts

Use the Find panel (Edit menu) to search among records for text that carries particular attributes, or lacks particular attributes.

To find records that contain (or do not contain) text formatted in a particular style or font, click Attributes… Cindex will display a panel through which you can specify the attributes that must be carried (or must not be carried) by the text you are looking for.

To find any text set in a particular style and/or font (or that must not be set in specified styles and font), click Attributes… to specify the style and/or font, but do not specify search text.
