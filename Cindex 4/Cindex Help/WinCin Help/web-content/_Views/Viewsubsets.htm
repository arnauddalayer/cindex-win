Viewing a subset of records

By default, Cindex displays all records in the index. Cindex can display different subsets of records, as if they were the only records in the index:

  • To view only the records added to your index during your current session with Cindex, choose New Records from the View menu. If the index contains no new records, this item is dimmed. Cindex always displays new records in the order in which you added them to the index.
  • To view only records that have a particular label attached to them, or records that have no label attached to them, choose Hide by Attribute from the View menu. This command is available only when you are working with a fully formatted view.
  • Cindex can form and display an arbitrary set of records as a group. A group is usually formed from a selection of records, or as the result of a search.

To restore the display of all records choose All Records from the View menu, or by clicking the button in the main toolbar.
