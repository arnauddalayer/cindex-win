Identifying the indexer

Cindex can include in each record information that identifies the person who made or modified the record.

To include such information in records, choose Preferences… (Edit menu) and enter up to four identifying characters at User ID. Cindex will attach the identifier to each record you make or modify (this information is not part of the record text).

If you want to use a different identifier in different sessions with Cindex, check Prompt for User ID. Each time you start Cindex it will display a panel through which you can enter your identifying initials.
