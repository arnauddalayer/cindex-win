Index templates

When Cindex creates a new index it can use the record structure and other standard settings (for sorting and formatting entries) from an existing index. To use one index as a model for another, you must first create a template. You can make a template in two ways:

To create a new index with attributes modeled on a template requires two steps:

  1. Make a template from an existing index. To do this, make sure the index is open and active, then choose Save As... from the File menu. Select Template from the Save as Type drop-down list in the dialog box.
  2. Make the new index from the template. Choose Open... from the File menu, then open the template (to make Cindex display template names in the dialog box, choose Templates from the Files of Type drop-down list). Cindex will ask you to provide a name for the new index it creates.

When you open a template with Cindex (through Open… from the File menu), Cindex will ask you to provide a name for a new index. When you provide the name Cindex will create the new index with the settings contained in the template.
