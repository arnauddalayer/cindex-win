Setting attributes of new indexes

Cindex provides each new index with default settings for many of its attributes. You can change most of these defaults, so that new indexes have the properties you want.

Cindex provides every new index with a default font and size in which all entries are displayed, except for parts explicitly marked for display in some other font. The default font is initially set to Arial, 12 point, though you can change the default at any time.

If you are working in a language other than English, choose Sort from the Tools menu to ensure that Cindex uses the appropriate collation rules for that language. Making this setting also ensures that Cindex will correctly lay out entries for right-to-left reading languages such as Arabic and Hebrew.

To set other default attributes of indexes, make settings through various panels noted below.

The settings you make in these panels become default settings only if no index is open for work. If an index is open, the settings you make are applied to that index.
  • To specify the structure of index records, choose Record Structure from the Document menu.
  • To specify how Cindex should recognize page references and cross-references, choose Reference Syntax from the Document menu.
  • To specify how entries will be sorted, choose Sort from the Tools menu.
  • To set margins and columns, choose Margins & Columns from the Document menu.
  • To set headers and footers, choose Headers & Footers from the Document menu.
  • To specify how entries should be grouped, choose Grouping Entries from the Document menu.
  • To set index style and layout, choose Style & Layout from the Document menu.
  • To specify heading typography and punctuation, choose Headings from the Document menu.
  • To specify how cross-references are formatted, choose Cross-References from the Document menu.
  • To specify how page references are formatted, choose Page References from the Document menu.
  • To specify text that will have special typography, choose Styled Strings from the Document menu.
