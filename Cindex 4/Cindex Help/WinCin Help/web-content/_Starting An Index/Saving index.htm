Saving changes to an index

To save changes to the active index, choose Save from the File menu.

As you add or modify index entries, Cindex automatically saves them in your document on disk. To allow you to work quickly, however, Cindex does not do this immediately for every small change you make. Cindex is initially set to save changes automatically every ten minutes; you can change the interval at which Cindex saves changes, or you can disable automatic saving.

To restore the index to the last-saved version choose Revert to Saved from the File menu. Cindex will discard any changes you have made since you last used Save, or since Cindex last saved the index automatically.

To save and close an index document, first make sure that the main index window is active, then choose Close from the File menu, or click the close box at the top right-hand corner of the window.

To save the active index under another name, and work on this new copy, choose Save As... from the File menu, then choose Index from the Save as Type drop-down list in the dialog box. Cindex closes the active index and opens the new copy.

To save a copy of the active index under another name, without closing the active one, choose Save Backup Copy... from the File menu.
