Labeling records

You can attach a label (a private tag) to any records you choose. A label serves as a marker you can use for any purpose you choose (for example to identify records that contain incomplete entries). A labeled record is displayed in a distinctive color of your choosing.

You can use up to 7 different labels to tag records (an individual record may have only one label).

To label a record or records in the main view, select them in the main document window then choose from the Labeled submenu of the Edit menu, or choose from the menu attached to the toolbar button (or right-click and choose from the contextual menu).

To remove a label from a record or records, select the record(s) then either:

  • Choose from the submenu the label you want to remove. This will remove only the chosen label from records that bear it. It will not affect any selected records that carry different labels.

  • Choose No Label from the menu. This will remove all labels from selected records.

If the record-entry window is open, you can label the current record by choosing from the Labeled submenu, or by choosing from the menu attached to the toolbar button.

When you add or change a label, Cindex by default does not mark the record as 'modified' (i.e., it does not update the date stamp and user identifier). To set whether a label change is considered a modification, choose Preferences… from the Edit menu and, under the Edit tab, click Label Sets Change Flag.

To set the colors in which you want labeled records displayed, choose Preferences… from the Edit menu then, under the View tab, double click the color cell for the label you want to change. Cindex will display a panel through which you can choose the color you want.

Labeled records are distinctively colored in draft views only; in the fully formatted view labeled records are normally not distinguished from others. To make Cindex distinguish labeled records in the fully formatted view, choose Preferences… from the Edit menu and, under the View tab, check Show in Fully Formatted View.
