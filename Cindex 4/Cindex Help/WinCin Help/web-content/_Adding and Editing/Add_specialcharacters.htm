Special characters

The characters < > { } ~ and \ have special significance in sorting entries. To include one of these reserved characters as a literal character in a record, place \ immediately before it. Thus, to use \ as a literal character you would type \\ in a record; to use ~ as a literal character you would type \~. When Cindex displays the index entry it shows only a single \ or ~.
