Using hot keys

Cindex lets you define a set of up to ten keys or keystroke combinations to which you can attach text that will automatically be inserted in a record when you hit the appropriate key or combination. You can also use hot keys to hold text or patterns that you might want when using Find… or Replace… to search for text.

By default, Cindex defines attachments for two hot keys: SHIFT F1 will insert the word See in a record; SHIFT F2 will insert the words See also.

To set up or review the hot keys and their associated text, choose Hot Keys… from the Tools menu.

To define a hot key, click (or tab to) a field in the left hand column, and enter the keystroke combination (e.g., CONTROL F1) you want to use (do not type the words as you see them in the list, simply enter the keystrokes). If you enter a forbidden keystroke or combination, such as a single letter, Cindex will modify it.

Choose your hot key combinations carefully so as not to conflict with normal key usage by Cindex or by other programs. When you have defined a hot key, that definition replaces any previously assigned use of the key or keys, and prevents Cindex (or any other active application program) from using the key while Cindex is active.

To attach text to a key, type it in the counterpart field to the right. You can also attach text by dragging it or pasting it from another window.

Text attached to hot keys can be styled (boldface, etc.) and can use the symbol font. Information about other fonts is not preserved. To include style or font attributes in the text attached to a hot key, you must paste or drag into the field the text that contains these attributes. The most convenient source of text will generally be a selection from the record entry window.
