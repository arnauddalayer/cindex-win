Accented letters and symbols

Index entries can contain any characters displayable by Windows.

Cindex understands the keyboard shortcuts used by Microsoft Word for entering accented characters.

You can also enter an accented letter (or any special character, such as a Greek letter or mathematical symbol) via the Character Map… from the Character menu.

To view the characters available in a particular font, choose the font by name from the Font drop-down list. To insert a character into a record, click in the record window at the place where you want to insert the character, then click the character in the Character Map to enlarge it and drag it into the record.

To deploy a special subset of Unicode characters from the Character Map (e.g., Greek letters, or mathematical symbols), do the following:

  • Check Advanced View.
  • From the Group dropdown list, choose Unicode Subrange.
  • From the list in the panel that is displayed, choose the character subset you want. This subset will be displayed in the main Character Map panel.

You can also enter any character in a record (or other places where you would provide text) by typing its Unicode value: hold down the ALT key then type the four-digit value on the numeric keypad.
