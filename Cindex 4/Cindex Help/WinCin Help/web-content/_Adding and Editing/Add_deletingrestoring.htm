Deleting and restoring records

To remove a record (or records) from the index, select them in the main document window then check Deleted in the Edit menu (or click the button in the toolbar). A deleted record is not physically removed from the index—it merely becomes hidden when you view a fully formatted index. In a draft or unformatted view, Cindex will identify a deleted record by displaying an x in the left margin beside it. Deleted records are never displayed in a formatted view.

To restore deleted records, select them in the main document window and uncheck Deleted in the Edit menu.

If you are working in the record window you can delete the current record by checking Deleted in the Edit menu. You can restore a deleted record by unchecking Deleted. The status line at the bottom of the record window displays “Deleted” if the current record is deleted.

You can remove deleted records permanently by choosing Compress… from the Tools menu. Deleted records cannot be restored after you have compressed the index.
