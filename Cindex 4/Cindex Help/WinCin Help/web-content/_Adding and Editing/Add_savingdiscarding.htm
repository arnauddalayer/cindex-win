Saving or discarding changes to a record

To save a new record and close the editing window, hit CONTROL RETURN (or ENTER on the numeric keypad). If you save a new empty record Cindex automatically discards it.

If you close the record window by clicking the close box at the top right, or by choosing Close from the File menu, Cindex will normally ask if you want to save changes to the record. You can change this behavior by customizing Cindex.

To save a record and leave a copy of it as the foundation of a new record, hit SHIFT PAGE DOWN or click .

To discard the contents and close the window, hit ESC.
