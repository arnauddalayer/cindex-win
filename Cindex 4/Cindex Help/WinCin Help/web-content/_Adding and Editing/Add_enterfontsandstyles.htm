Fonts and styles

Except for parts of entries explicitly marked for display in a particular font, all entries are displayed in a standard font. When the main window for an index is active, you can change the standard font and type size by choosing from the appropriate drop down list in the main toolbar.

When Cindex displays records in the record-entry window it normally uses the font size set for the main view. To override this, choose Preferences… from the Edit menu, click the View tab, then set the size at Entry View.

When you are working in the record window, you can set any part of an entry in a font of your choosing. To do this, select the text you want to change, then choose from the font menu in the main toolbar.

To restore to the standard font any segment of text set in some other font, select the text, then choose Deafult Font from the Character menu or right-click and choose Default Font from the contextual menu.

The default style for an index is plain text. You cannot apply another style to a complete index, but through format settings you can apply a style automatically to particular elements of entries. For example, Cindex can display all main headings in boldface capitals, regardless of how you entered the headings in records.

You can apply styles to selected text in the record window. To apply a style, select the text, then choose from the Character menu, or right click in the record window to display a contextual menu. You can apply multiple styles to the selection. If you choose a style when no text is selected, that style will be applied to characters you subsequently type. You can remove all styles from a selection by choosing Plain from the Character menu or the contextual menu. To remove just a single style from a selection that contains several styles, choose that style from the menu.

If you apply the small caps style, Cindex will display reduced-size text in the record window, with no change of case. In the main index window the text will appear correctly: lowercase letters will be displayed as small uppercase letters.
