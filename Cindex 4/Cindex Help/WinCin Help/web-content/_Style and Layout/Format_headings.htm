Heading typography and punctuation

In a fully formatted view Cindex can display each level of heading in a specified font and style, and can append punctuation to headings that have no immediately following references. Cindex can also suppress entirely the display of headings at particular levels.

To specify the font and style in which headings will be displayed in the formatted index, choose Headings... from the Document menu. Click the tab for the heading level that you want. If you have specified (by setting Record Structure) that index records contain a required last subheading, the settings you make on the last tab will be used to format the special subheading, regardless of its actual depth in a record, and the title of the tab will contain an asterisk (*) to remind you of this.

Choose from the Font drop-down list the font in which you want the heading displayed. The first item in the menu (<default>) represents the standard font in which the index is displayed.

Click Style... to specify the style in which you want the heading displayed. If you choose Title Case for capitalization, Cindex capitalizes words in headings as follows:

  • The initial letter of every word is capitalized, unless preceded by ~.
  • Every letter after the first in a word is set in lowercase, unless preceded by ~.
  • Words in the list for smart flipping (Smart Flip Words… from the Document... menu) are not capitalized.

To make Cindex show leading text immediately before the heading provide that text at Leading Text.

To specify closing punctuation or other characters that Cindex will append to any heading level that has no immediately following page reference or cross-reference, supply the punctuation at Trailing Punctuation.

To suppress the display of the heading in the fully-formatted index, check Suppress. Cindex will suppress the heading and any associated page references and cross-references.

If you set heading typography when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
