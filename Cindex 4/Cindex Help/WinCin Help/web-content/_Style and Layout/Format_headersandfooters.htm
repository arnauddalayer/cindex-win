Headers and footers

To set headers and footers for your index, choose Headers and Footers... from the Document menu.

If you have chosen to format the index with facing pages (Margins and Columns, above) Cindex provides separate headers and footers for left (even) and right (odd) pages. Otherwise it uses the same headers and footers for odd- and even-numbered pages. Click the tab for the header or footer settings you want to inspect or set.

Each header or footer can contain three text segments, Left, Centered and Right. Enter the text you want to appear in each of these positions. In any text segment the following characters stand for special items, and are replaced by the appropriate item when Cindex prints the header or footer:

  • @ denotes the current date and time.
  • # denotes the current page number.
  • % denotes the name of the index.

To use @, # or % as a literal character in a header or footer field, place a \ immediately before the character (as in \@).

Cindex prints each header or footer in the font chosen from the Font drop-down list in the size specified in the Size box, and in the style set through Style... The first item in the Font menu <default> represents the standard font in which the index is displayed. You can choose a size from the Size drop-down list, or type in a size of your choice. To make Cindex use characters of the size used in index entries, set the size to 0.

If Cindex displays a page number in a header or footer (i.e., you have included #), First Page specifies the number printed on the first page. Choose from Number Format how you want the number formatted—in Arabic or roman numerals.

If Cindex displays a date in a header or footer (i.e., you have included @), the date appears in the format chosen in Date Format. The Short and Long formats are those specified through the Date tab of the Regional Settings Control Panel. Cindex will add the time to the printed date if you check Add Time. The time format is that specified through the Time tab of the Regional Settings Control Panel.

If you are using facing pages, and want left and right pages to have identical headers and footers, set the header and footer for one page, then click Copy. This is dimmed if left and right settings are identical.

If you set Headers and Footers when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
