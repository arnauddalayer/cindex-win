Line spacing

Cindex will space lines automatically, at an interval appropriate for the font and type size in which entries are displayed, or at any fixed interval you specify.

To specify the spacing of index entries, choose Style & Layout... from the Document menu.

Under Line Spacing, choose from the menu the spacing you prefer: Single, 1.5 Lines or Double.

To let Cindex set the separation of lines appropriately for the font and size of text you are using, check Auto. Cindex displays that separation in whatever unit you choose from the Unit drop-down list. To fix the spacing in the unit shown in the drop-down list, uncheck Auto, then type the value in the box.

Cindex can insert extra blank lines between main headings, and between alphabetical groups of entries.

  • To insert extra lines between main headings, set the number of lines as Extra at Main Headings.
  • To insert extra lines between alphabetical groups of entries, set the number of lines as Extra at Alphabetic Groups.

Cindex inserts extra lines only when displaying the index in fully formatted view.
