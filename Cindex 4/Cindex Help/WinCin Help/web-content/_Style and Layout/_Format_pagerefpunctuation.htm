Punctuating page references

Use the Page References panel (Document menu) to specify the punctuation of locators.

Cindex always displays page references immediately after the text of the entry to which they belong. Cindex inserts punctuation of your choice before and after a set of page references. If you wish, Cindex will provide different leading punctuation depending on whether a single page reference follows a heading or multiple references follow a heading.

Provide at Before Single the punctuation Cindex should insert between a heading and a single following page reference.

Provide at Before Multiple the punctuation Cindex should insert between a heading and several following page references.

Provide at After any punctuation Cindex should append to the last page reference from a heading.

To make Cindex right-justify page references within the width of a column, check Right Justify. When references are right-justified you can have the leader space blank, or filled with a dotted leader. To use a dotted leader, check Dot Leader.
