Placing and punctuating cross-references

Use the Cross-References panel (Document menu) to specify the formatting and punctuation of cross-references from main headings and subheadings.

Click the From Main Heading tab to show settings for main headings. Click the From Subheading tab to show settings for subheadings

Cindex can punctuate See and See also references differently.

Enter under Prefix the punctuation (including any spaces) you want to separate each kind of cross-reference from its heading or preceding page references.

Enter under Suffix any trailing punctuation (e.g., a period, or a closing parenthesis) that should follow the last cross-reference made from an entry.

See references, which direct a reader to other entries in the index, generally appear immediately following the headings from which they are made. Cindex can place a See reference either run on from the heading, or as a subheading. Choose the arrangement from the first pop-up menu under Position.

See also references are made from entries that also carry page references, and their positions depend upon the style of the index and the preference of the publisher. You can make the following settings for See also references:

Heading (always)

References from a heading follow the heading (or any preceding page reference) directly, without a line break. Cindex is initially set use this placement of “See also” references.

Heading (no page)*

References from a heading runs on from the heading, without a line break, but only if there are no preceding page references. If there are preceding page references, the See also references are formatted as a subheading.

First Subheading

References from a heading are gathered together and displayed as the first subheading, above any regular subheadings that carry page references.

Last Subheading (always)

References from a heading are gathered together and displayed as the last subheading, below regular subheadings if there are any.
Last subheading (conditional)* References from a heading are gathered together and displayed as the last subheading, below regular subheadings if there are any. If there are no subheadings, the cross-references are run-on from the page references without a line break.

*When a See also reference may be run on from a heading or placed as a subheading, depending on context, Cindex supplies Prefix and Suffix punctuation only when the cross-reference is run on from the heading.
