Formatting cross-references

Cindex can place cross-references in virtually any position you want. You can specify separately the placement of See and See also references, for cross-references from the main heading and for references from subheadings.

Cindex can provide different opening and closing punctuation for See and See also references, separately for main headings and subheadings. Cindex will also apply styles of your choosing to the introductory phrase and to the body of the cross-reference.

To specify how cross-references should be placed and punctuated, make settings in the From Main Heading tab or the From Subheading tab.

To specify style and capitalization to be applied automatically to the introductory phrase of a cross reference, click Prefix Style.

To specify style and capitalization to be applied automatically to the body of a cross reference, click Body Style. If you choose Title Case for capitalization, Cindex capitalizes words in headings as follows:

  • The initial letter of every word is capitalized, unless preceded by ~.
  • Every letter after the first in a word is set in lowercase, unless preceded by ~.
  • Words in the list for smart flipping (Smart Flip Words… from the Document... menu) are not capitalized.

Check Alphabetize Multiple References to permit Cindex to alphabetize a series of cross-references made from the same heading. If this is unchecked, cross-references are displayed in the order in which they appear in records.

To suppress entirely the display of cross-references in the fully-formatted index, check Suppress All References.

To prevent Cindex applying an automatic style to the cross-reference prefix if that style is already applied to the body of the cross-reference, check Don't Apply Prefix Style if it Matches Existing Body Style.

If you make settings when no index is open for work, the settings become the defaults that Cindex uses for new indexes when you create them.
