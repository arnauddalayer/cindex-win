Setting margins and columns

To set margins and columns for your printed index, choose Margins & Columns... from the Document menu.

First choose from the Unit drop-down list the unit you want to work in.

To set margins, enter the sizes you require in the boxes marked Top, Bottom, Left, and Right. If you want the index laid out with facing pages (with inside and outside margins), check Facing Pages. Use the Unit drop-down list to see your settings displayed in any unit.

Cindex will print pages with up to six columns of entries. To specify the number of columns, choose from the Columns drop-down list. Set Gutter to the space you require between columns. Use the Unit drop-down list to see your setting displayed in any unit. When you choose more than one column, Cindex displays text in a column of the correct width on-screen, but does not show side-by-side columns. These appear only on the printed page.

When printing an index in the fully-formatted view only, Cindex can repeat at the top of any new column or page the heading from any entry broken at the foot of the previous column or page. When it repeats the broken heading Cindex can append some phrase (such as “continued”) to indicate that the entry was broken. Page and column breaks are not visible in the screen display, so Cindex does not show on the screen any repeated headings from entries that will straddle a column or page break.

To repeat no headings, click Never. To repeat a broken heading after a break, check At New Page, or At New Column. Choose from the To Level menu the lowest level heading to be repeated after a page or column break. Enter at Append any phrase you want appended to a repeated heading. Click Style... to specify the style in which Cindex should print appended text.

Choose from the To Level pop-up menu the lowest level of heading that you want repeated after a break. For example, to have only main headings repeated, but not subheadings, choose the first item.

To make Cindex print some identifier after the repeated heading or subheading, supply the text at Append. To specify the style in which the appended text should be displayed (italic is the default) click Style… Cindex will display a panel through which you can set the style and capitalization.

If you set Margins and Columns when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
