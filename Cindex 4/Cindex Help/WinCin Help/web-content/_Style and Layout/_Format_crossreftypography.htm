Cross-reference typography

Use the Cross-References… panel (Document menu) to specify the type style(s) in which cross-references should be displayed in the formatted index.

Cindex will automatically apply a style and capitalization you designate to the introductory phrase of a cross-reference, and will apply the same or a different style to the body of the cross-reference. When Cindex is first installed it is set to display the introductory phrase in italics, and the body in plain text.

Click Prefix Style… or Body Style… to show a panel through which you make settings for the introductory text or the body of a cross-reference.

Choose the styles and form of capitalization you want. In the prefix style you can set Automatic capitalization to make Cindex capitalize the phrase according to preceding punctuation: Cindex will automatically capitalize the first word of the prefatory phrase if the cross-reference is formatted as a subheading, or the preceding punctuation mark (with any intervening space) is a period; Cindex will set the initial letter to lowercase if the preceding punctuation is a comma (with any intervening space) or an opening parenthesis.

To prevent Cindex applying an automatic style to the cross-reference prefix if that style is already applied to the body of the cross-reference, check Don't Apply Prefix Style if it Matches Existing Body Style.

Cindex does not apply styles or capitalization to the body of any cross-reference that begins with a word you have identified as introducing a general cross-reference.
