Indentation

To specify indentation of headings and runover lines, choose Style & Layout... from the Document menu.

Under Indentation, choose from the Type drop-down list how Cindex will form heading and runover indents (in the fully formatted view only):

None

Cindex provides no leading or runover indentation.

Incremental

Cindex indents the start of each subheading by a specified amount that depends upon the subheading level; for any runover lines Cindex adds an additional amount to the lead indent. You specify the runover indentation and the lead for the first level of subheading, and Cindex calculates all other indents automatically.

Fixed

Cindex indents headings and runover lines by amounts that you specify separately for each level of heading. This method provides the greatest control over indentation, but requires more settings than others. Choose Fixed to use this method. If you choose fixed indentation Cindex displays a popup menu through which you choose the heading(s) for which you will set indentation.

Special Last

This method is like standard incremental indentation, but the lowest level of subheading in an entry (regardless of its actual level) has fixed lead and runover indents. This method is designed for indexes in which the body of an entry must be indented by a fixed amount regardless of its placement under a main heading or a subheading

If you chose Fixed indentation, click the tab corresponding to the heading level for which you want to specify an indent. Do this for all levels of heading in the index.

If you use Special Last indentation, click the Incremental Indent tab to specify the indent sizes for headings that will be indented automatically, then click the Last Indent tab to set the last indent.

Specify a Lead indent and a Runover indent in one of the following kinds of units:

  • Click Em spaces to use em spaces for the font in which the index is displayed.
  • Click Absolute to use an absolute value in the unit shown in the adjacent drop-down list.

How Cindex uses the indent values you specify depends upon the indentation method you have chosen.

  • With Incremental indentation, Cindex indents each heading by its level x Lead indent; for runover lines Cindex adds the Runover indent to the lead indent.
  • With Fixed indentation, Cindex indents each heading and runover line by exactly the Lead and Runover amounts you have specified for each level.
  • With Special Last indentation, Cindex indents all headings but the lowest by the amounts set for incremental indentation; the lowest subheading is indented by exactly the amounts specified for Lead and Runover.
