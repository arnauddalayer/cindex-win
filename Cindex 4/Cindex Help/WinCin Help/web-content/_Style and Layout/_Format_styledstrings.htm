Styling text automatically

You can designate certain words and phrases (e.g., versus) to which Cindex will automatically apply a style or styles when it encounters them in index entries. To designate words or phrases, and their associated styles, choose Styled Strings... from the Document menu. The list shows the words and phrases to which Cindex will automatically apply a style.

To add a word or phrase, click Add, then type it in the box Cindex opens in the table. Check the styles you want applied to it.

To change the styles applied to a word in the list, click to select it then check the styles you want applied to it.

To edit a word or phrase in the list, double-click it.

To delete a word or phrase, click to select it in the list, then click Delete.

Cindex applies automatic styles only when displaying entries in a fully-formatted view.

If you set Styled Strings when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
