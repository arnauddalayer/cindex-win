Grouping entries

When it displays or prints a fully-formatted index, Cindex can set-off different alphabetical groups of entries by printing a title above the group (typically the letter of the alphabet under which entries in the group fall). To specify how Cindex should group entries, and what text it should display above a group, choose Grouping Entries… from the Document menu.

At Text enter any title text you want to have displayed above the group (if you supply no text, Cindex displays no title). Use the character %, on its own or as part of a longer text sequence, to stand for the leading letter of the group of headings. When Cindex displays the title it will replace the % with the appropriate letter or symbol.

Cindex always treats each letter of the alphabet as belonging to a distinct group, but lets you choose how it groups entries beginning with numbers of symbols. Use the Grouping Method drop-down list the method you want it to use:

[@] [#] [$]… [1] [2] [3]…

each leading symbol or numeral forms a distinct group

[@#$…] [1] [2] [3]…

all leading symbols form one group; each numeral forms a distinct group

[@] [#] [$]… [123…]

each leading symbol forms a distinct group; all numerals form one group

[@#$… 123…]

all leading symbols and numerals form a single group

[@#$…] [123…]

all leading symbols form one group; all numerals form one group

Enter at All Numbers Title, etc., the text to be displayed at the head of entries that are grouped generically under numbers, or symbols, or numbers and symbols.

Cindex prints each title in the font chosen from the Font drop-down list and in the style set through Style... The first item in the Font list <default> represents the standard font in which the index is displayed.

If you make settings when no index is open for work, the settings become the default ones that Cindex uses for new indexes when you create them.
