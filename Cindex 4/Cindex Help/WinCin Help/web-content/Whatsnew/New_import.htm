New features: importing index records

Macrex
Cindex now provides much richer capabilities for importing index records in Macrex backup format, including translation of the special character sequences Macrex uses to represent unicode characters.

SKY Index
Cindex can now import index records in the export format used by SKY Index 8 (in addition to the previously available formats for SKY Index 6 and 7), providing full support for unicode and fonts.
