Introducing Cindex 4

Cindex 4 makes it easier than ever to produce the highest quality indexes in the shortest time. If you are a Cindex user upgrading from Cindex 3, review the following sections to learn about new features. New users of Cindex can skip to the Cindex Guide.

Adding and editing records

Easily demote headings.
Split headings and extract components into new records.
Improved handling of styles and fonts when pasting or dropping text.
Enhanced capabilities for adjusting page references.

Displaying the index

New option for displaying headings in title case.
Automatic detection and suppression of duplicated references.

Checking and searching among entries

New comprehensive checks for correctness and consistency of entries.
Improved searching by date.
Improved search and replace for text attributes.
Powerful new ready-made patterns for Find and Replace.

Sorting entries

Sort text as if it were other text (e.g., St. as Saint).
Improved sorting of complex locators.

Exporting the formatted index

Richer entity markup in XML documents.
New option for specifying style names in exported documents.

Importing index records

Richer import import of records from Macrex, including unicode characters.
Richer import of records from SKY Index.

General

Wide-ranging improvements in general performance.
