New features: exporting formatted indexes

Index-Manager
Cindex provides a new export option to save an index in the format required by Index-Manager for embedding index entries.

XML text
When exporting a formatted index as an xml document, Cindex now provides a tag for the body text of an entry (i.e., the entry text before any page references or cross-references) as an explicit element.

SGML Text
When exporting a formatted index as a tagged text document, Cindex now provides an option to represent accented letters and other unicode characters directly, rather than encodeing them as ASCII character sequences.

User-specified style names in exported documents
When exporting a formatted index in Rich Text Format, or for InDesign or for QuarkXPress, Cindex used the names you had assigned to index fields—'Main', 'Sub 1', etc., as the names of paragraph styles in the exported document. A new option available when you export a formatted index enables you to set the names of paragraph styles in the document. The old names are still used by default.
