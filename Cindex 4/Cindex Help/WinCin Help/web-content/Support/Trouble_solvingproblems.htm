Solving problems

If you need help solving a problem you're having while using Cindex, look below for a solution. If you can't find what you're looking for, try searching help, or click the link at the bottom of the page for further information.

If you experience a problem with the performance of Cindex, check that you are using the most up-do-date version. From time to time we provide updates. These are designed to correct problems and improve performance. You can set Cindex to check automatically, and notify you, if an update is available. You can also check by choosing Check for Update from the Help menu.
