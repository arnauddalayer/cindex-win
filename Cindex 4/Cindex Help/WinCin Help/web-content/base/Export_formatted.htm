Saving formatted indexes

Cindex can save an index in several different formats, for use by typesetting systems or other document preparation systems.

To save an index as formatted document, choose Save To… from the File menu. From the File Format pop-up menu choose the type of document you want to make.

Choose from among the following types of formatted document:

Plain Text

Entries are formatted as you would view them on the screen, but contain no information about fonts or type styles. By default Cindex encodes text using the traditional Windows ANSI character set. This character set cannot represent all Unicode characters, so the encoding may lose information. To ensure that all characters are encoded, you can export Plain Text using UTF-8 encoding. To do this choose Preferences… from the Cindex menu, then click the Export tab. When Cindex exports the document as UTF-8 encoded text, it places the standard UTF-8 signature at the start of the file. This enables it to be read correctly by most text-handling programs on Windows and Mac.

Rich Text Format (RTF)

RTF is a widely-used interchange format for text documents. Each level of index heading in the RTF document has a unique ‘style’ attached to it. Cindex sets the initial properties of the document and its styles to reflect the settings you have given the index (number of columns, indentations of headings and runover lines, etc.), so that when the document is imported into a word-processor or page-layout program it will look much like it did when displayed by Cindex. The appearance of the index can be changed within a word-processor or page-layout program simply by re-defining the attributes of the styles associated with different levels of heading.

QuarkXPress

QuarkXPress is a publishing program that uses its own of system of tags, called XPress tags, to identify styles that belong to different elements of a document. Cindex can save a formatted index as an XPress tagged document. Each level of heading has a unique ‘style’ attached to it, and carries the attributes that belong to that heading. These attributes can be changed when the document is read by QuarkXPress.
Adobe InDesign InDesign is a document layout program that uses its own system of tags to identify different elements of the document. Cindex can save a formatted index as an InDesign tagged document. Each level of heading has a unique ‘style’ attached to it, and carries the attributes that belong to that heading. These attributes can be changed when the document is imported by InDesign.
Index-Manager Index-Manager is a program designed for embedding index entries in Microsoft Word and Adobe InDesign documents. Cindex can save a formatted index as an embedding-ready document that Index-Manager can import directly.

XML Tagged text

You can save fully formatted indexes as XML documents. These are marked-up plain text documents in an open format, and can be interpreted by any application program that can parse XML. Cindex can produce tagged-text documents of various kinds.

SGML Tagged text

Many document management systems work with documents in which the elements (such as headings and subheadings) and text attributes (such as boldface) are identified by tags that system uses when laying out the formatted document. Cindex can produce tagged-text documents of various kinds.

When Cindex prepares an index as a document for a typesetting system or a word-processor, it always formats entries as they would appear in a fully formatted view (even if the current view is set to some draft format). Cindex includes in the document most of the information needed to make the index look like it did when viewed as a fully-formatted index on the screen.

Cindex normally saves all of the index that is accessible in the current view (either all records, or a group). To modify this behavior and others, click Options in the Save panel before you save the index.

If your index entries have used a special last subheading, when forming the exported entry Cindex formats the last subheading as though it were at the deepest level the index can accommodate, and applies the formatting associated with that level.
