Printing

Setting Print Format

The overall layout of printed pages is determined by settings you make for margins and columns.

To set the paper size and orientation for printing, choose Page Setup... from the File menu. If you wish, you can also set margins here.

Printing Index Entries

To print index entries, choose Print... from the File menu.

In addition to the standard printing options available to all Windows application programs, Cindex lets you choose which entries to print:

  • Click Selection to print selected entries (this option is dimmed if no entries are selected).
  • Specify in From and To the starting and ending entries of any range you want to print. You can specify these entries by the beginning text of their headings, or by their record numbers.

Cindex also lets you print odd and even pages as separate sets. Choose from the Print drop-down list.

Choosing a Printer

If you have more than one printer accessible to your computer, you can specify the one you want Cindex to use. To do this choose Page Setup... or Print... from the File menu, then choose the printer from the menu that displays printer names.

To install or configure a new printer, choose Printers and Faxes from the Windows Start menu.
