Managing cross-references

Cindex provides two powerful capabilities for managing cross-references:

  • Cindex can generate cross-references automatically, so that the existence of a specified entry can lead automatically to the creation of an appropriate cross-reference. How to generate cross-references.
  • If a cross reference points to too few target entries, Cindex can automatically convert the cross reference to a fully qualified entry. How to convert cross-references.
