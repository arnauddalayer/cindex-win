Options for saving formatted documents

To set options for saving formatted, click Options… in the panel Cindex presents when you choose Save As… from the File menu.

Settings you make here control how Cindex saves formatted entries in a document.

  • Items under Range to Save define the collection of records to be saved. For more information on this see Specifying the scope of a command.
  • When saving documents in Rich Text Format, or in the formats required by QuarkXPress or InDesign, Cindex can define leading indents to subheadings in one of several ways. Click the appropriate button beside Subhead Indent to specify whether indents should be defined as an attribute of the style attached to the subheading, by ‘tab’ characters (one for each level of subheading), or by a character of your choice (one is emitted for each level of subheading). You can specify default settings by choosing Preferences… (Edit menu) and clicking the Formatted Export tab.
  • By default, Cindex identifies levels of heading in the exported index using the names you set for record fields in the index. To use different names, click Paragraph Style Names… Cindex will present a panel through which you can set, and enable, the names you want to use for headings in the exported document. The names you provide are specific to the type of document you are creating (e.g., QuarkXPress or InDesign). Cindex remembers the names, even if you are not using using them.
