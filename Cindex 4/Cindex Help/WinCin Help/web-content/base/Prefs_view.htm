View preferences

To inspect or change settings that control how Cindex displays entries in the main index view and the record-editing window, choose Preferences... from the Edit menu, then click the View tab.

  • Settings under Default Font establish the font used initially when you create a new index.
    Choose the font by name from the drop-down list.
    At Main View set the size of type Cindex will use to display entries in the main window.
    At Entry View set the size of type Cindex will use to display entries for editing in the record window. If this is 0, Cindex uses the size you have set for the main window (above).
  • Settings under Label Colors control the display of labeled records.
    To choose the color Cindex uses to represent a particular label, double-click the colored well that corresponds to the label. Cindex will display a panel through which you can pick the color you want.
    By default, Cindex displays labeled records in their signature colors only in a draft view. To make Cindex use the colored display in a fully formatted view, check Show in Formatted View.
