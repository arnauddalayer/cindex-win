Embedding entries in other documents

Cindex can directly embed index entries in Microsoft Word documents or documents from any other word-processor that uses the standard Rich Text Format (RTF) definitions of index entries.

To embed entries you should first set the index to draft view. Cindex does not prepare entries for embedding if you are working in a fully formatted view.

Drag (or copy) entries into your word processor document at the places referred to by the entries.

Cindex formats embedded entries as hidden text. If you want to see the text that has been embedded be sure to set your word-processor to show hidden text.

Entries embedded by Cindex have the following properties:

  • They carry no page references (which will be supplied by Microsoft Word, or whatever word-processor you are using), but they do carry information about how the page references should be formatted (as boldface or italic, etc.—make these settings by choosing Page References… from the Document menu). For your own purposes you can place page references in the locator fields or records, but Cindex will not transfer them during embedding.
  • If the locator field contains the character you use to define a range of pages, either alone or as part of a real page reference, the embedded index entry will refer to a fictitious bookmark ("$$$") that you should later replace with a real bookmark to indicate the range of text that should be represented by the page range in the index entry.
  • The embedded index entries carry any cross-references from the locator fields of records. These will be formatted in the style you have previously specified by choosing Cross-References… from the Document menu.
  • Each heading in the index entry normally carries a sort key that is used by Microsoft Word when it arranges the entries in the formatted index. Cindex constructs this sort key (which will not appear in the formatted index) according to the sort rules you have specified for the index. To take maximum advantage of Cindex's powerful alphabetizing capabilities, before you embed entries you should ensure that they are sorted as you would wish them to appear. If you do not want Cindex to embed a sort key along with the entry, you can disable this by choosing Preferences… from the Edit menu. The setting is on the Formatted Export tab.
