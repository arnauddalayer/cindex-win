Adding an offset to page references

Use the Alter References panel (Tools menu) to add an offset (positive or negative) to specified page references in records.

Follow these steps:

  1. Under Scope choose whether you want to adjust All References accessible in the current view, or those falling within a specified range. To set a range click Range, and enter the lowest and highest page numbers that should be adjusted. Be sure to enter the numbers as they exist before adjustment. If you specify only an upper limit Cindex assumes the lower limit to be 1. If you specify only a lower limit, Cindex assumes the upper limit to be all higher references.
    If your locators are compound references, (e.g., 23:14:2 or 34[2]:25–27), Cindex will normally adjust only the first component in Arabic numerals. To make adjustments to other parts of compound references, you can provide at Matching Pattern a pattern to specify leading parts of references that Cindex should skip to reach the single element it should adjust.
  2. Under Action click Adjust Only.
  3. Specify in Adjustment the offset to be added to (or subtracted from) every reference that falls within the range.

References needing adjustment may be intermixed in a record with others that do not need adjustment; Cindex disregards references that fall outside the range you have specified.

Cindex ignores references in Roman numerals, cross-references, and any references that contain no Arabic numerals.

If an adjustment would result in a page reference being less than 1, the reference is removed from its record.
