XML markup tags

Choose Markup Tags… from the Tools menu to view and edit sets of tags that Cindex uses to mark-up formatted documents. Click the XML tab, then choose the tag set from the pop-up menu.

If you are defining your own tags, note the following:

  • Each opening tag must have a matching closing tag. You specify only the opening tag; Cindex automatically generates the closing one.
  • A tag name may not contain any of these characters: < > & ' "

Structure Elements

These identify the different structural elements of an index. Cindex provides a tag to denote the index document (Index), a tag to denote an alphabetical group of records (Group), and a tag to denote the title to a group of records (Heading). Additional tags identify different levels of heading, and (optionally) the body text of an entry, page references, and cross-references.

Check Identify Level with Attribute to make Cindex identify the main heading/subheading level with a numerical attribute.

Check Nest Subheadings to make Cindex nest subheadings hierarchically under headings.

Check Tag Individual References to make Cindex apply the page reference tag or the cross-reference tag to each reference separately, rather than to the group of references in an entry.

Check Suppress Reference Leader & Trailer to make Cindex suppress the leading and trailing punctuation that it would normally place before and after page references and cross-references.

Other Elements

Style tags identify type styles, such as italic and boldface. Cindex will embed style tags in the document where the index contains styled text.

Font tags are used to denote changes of font. The first tag in the set denotes the default font in which the index is displayed in the main document window. Subsequent tags are used to identify any other fonts used in the index.

When Cindex emits a tag to identify a font, it can emit the unadorned tag, or can add an attribute that identifies the font, either numerically, or by name. To make Cindex add the identifying attribute click ID (for numerical identifier) or Name (for the name of the font).
