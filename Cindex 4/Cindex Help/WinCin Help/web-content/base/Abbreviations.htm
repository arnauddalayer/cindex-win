Managing abbreviations

Saving and loading abbreviations

Cindex works with one active set of abbreviations, and when you define a new abbreviation Cindex always adds it to the active set. You can maintain multiple sets of abbreviations for use on different occasions. Cindex keeps track of the set you last used, and automatically opens it when you start a new session.

To load and use an existing set of abbreviations, choose Load… from the Abbreviations submenu in the Tools menu.

To start a new set of abbreviations, choose New… from the Abbreviations submenu in the Tools menu.

To close the active set of abbreviations, choose Close from the Abbreviations submenu in the Tools menu.

Viewing and editing abbreviations

To edit abbreviations choose Edit… from the Abbreviations submenu in the Tools menu.

To add a new abbreviation, choose New Abbreviation from the Edit menu. Cindex will display, at the bottom of the current list, the name 'NEW'. Change this to the name you actually want to use. The name may contain up to eight characters, except a space and these punctuation marks:

( ) [ ] " “ ” ' ‘ ’ , . ; :

When you have named the abbreviation hit TAB (or click to the right of the name) to enter the text to be represented by it. To apply a style to any part of the text, select the text and choose the style from the Character menu.

You can also define abbreviations directly in records as you work on them.

To edit an existing abbreviation, simply change its name or the text attached to it.

To delete an existing abbreviation, select both its name and the text associated with it (or triple-click the line), then choose Clear from the Edit menu (or hit DELETE).

You can view abbreviations sorted alphabetically by the expanded phrase, or sorted alphabetically by name. To change the order click the title to the column on which you want to sort.

When the editing window is active, you can save your abbreviations by choosing Save or Save As… from the File menu. (If you have unsaved changes to abbreviations, Cindex will always ask you, before quitting, if you want to save the changes.)

Cindex remembers abbreviations after you close the editing window.
