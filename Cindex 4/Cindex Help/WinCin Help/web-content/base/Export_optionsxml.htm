Options for saving XML records

To set options for saving XML records, click Options in the panel Cindex presents when you choose Save To from the File menu.

Settings you make here control how Cindex saves index records in the exported document.

  • Items under Scope define the collection of records to be saved. For more information on this see Specifying the scope of a command.
  • Cindex does not normally save deleted records. Check Include Deleted Records to include deleted records in the saved document.
  • Cindex normally saves records in record number order. Uncheck Write in Record Number Order to make Cindex put records in the archive in the order which they appear in the current view.
