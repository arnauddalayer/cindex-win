General preferences

To specify general settings that govern the way Cindex behaves, choose Preferences... from the Edit menu. Settings you make under the General tab control the overall behavior of Cindex.

  • To make Cindex occupy the full screen when it starts up, check Maximize Cindex.
  • Check Prompt for User ID if you want to be prompted for your identifying initials, which will be attached to each index record you add or change. (If you don't often change your identifier, you should leave this unchecked, and set the identifier as described below.)
  • You can set Cindex so that whenever you start it, it will open the last index you worked on. To make Cindex open the last index check Open Last Index.
  • Cindex can save your changes to an index automatically at defined intervals. To make Cindex do this, set Save Index Every ... to the interval, in minutes, at which you want Cindex to save your index automatically to disk. To disable automatic saving of the index, set the interval to 0.
  • To set Cindex so that the main window of each index it opens occupies as much as possible of the screen, check Maximize Indexes on Opening.
  • To provide a user identifier that Cindex will attach to each record you make or change, enter up to four characters at User ID.
  • To set Cindex so that it will notify you atomatically when an update is available, check Automatically check for updates. Cindex requires an active internet connection when it checks for updates. You can also check manually at any time by choosing Check for Updates from the Help menu.
  • To specify a folder that Cindex will use by default when creating indexes or opening indexes, click Default Folder. Cindex will display a dialog box through which you can choose (or create) a folder that it will use the first time your open or save a document in each session.
