Generating cross-references automatically

Cindex can generate cross-references automatically, so that the existence of a specified entry can automatically lead to the appearance of a cross-reference.

To generate automatically a set of cross-references to specified headings in your index, you must build a special reference index that identifies the target entries and the cross-references that should be made to them.

Each record in the special index should be organized as follows:

  • In the main heading field enter the text that will be the main heading of the cross-reference.
  • In the first subheading field enter any subheading from which the cross-reference will be made.
  • In the locator field enter the text of the target entry, without any prefatory See or See also. Cindex will use the appropriate prefix when it forms the cross-reference.

To use the special index to generate cross-references for the active index, choose Manage Cross-References... from the Tools menu, then choose Generate from File….

From the dialog box choose the special index that is the source of potential cross-references. Cindex normally generates both See and See also references, as appropriate. To generate See references only (omitting any that would be See also references), check “See” References Only.

Cindex keeps track of records that have been generated automatically. You can find these by choosing Find… from the Edit menu; you can count them by choosing Count… from the Tools menu; you can remove them by choosing Compress… from the Tools menu.

If you modify the contents of a generated record, Cindex no longer considers it to have been generated automatically.
