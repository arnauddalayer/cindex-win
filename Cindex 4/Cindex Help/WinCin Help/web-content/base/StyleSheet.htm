Style sheets

You can gather together and save in a “Style Sheet” the collection of settings that together determine the layout of pages and index entries. You can later apply the style sheet to any index. You can keep different style sheets for different kinds of indexes.

The style and layout settings for an index are always saved with it. To save a style sheet, choose Save Style Sheet... from the File menu. Cindex saves in the style sheet the current values of these settings made via the Document menu:

Margins & Columns

Headers & Footers

Grouping Entries

Style & Layout

Headings

Cross-References

Page References

Styled Strings

The style sheet also contains information about the default font used by the index, and the size of text.

To use a saved a style sheet, choose Open… from the File menu, then select the sheet you want to open.

If you load a style sheet when no index is open for work, the settings in the style sheet become the default ones that Cindex uses for new indexes when you create them.
