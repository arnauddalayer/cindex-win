Creating a spinoff index from a group

Follow these steps to create a new index from a group of records in the parent index:

  1. Activate the main view window for the parent index.
  2. Set the view to show the group for a spinoff index (choose the group from the View menu).
  3. Create a new index for the group. Do not be concerned about setting the right record size—Cindex will re-size the index if necessary when it adds records to it.
  4. Select the records in the group in the parent index (choose Select All from the Edit menu).
  5. Drag the records to the main window of the spinoff index.
    Alternatively, you can Copy the records from the parent index (choose Copy from the Edit menu), then paste them into the spinoff index (choose Paste from the Edit menu).
  6. Repeat steps 1 through 5 for each group that contains records for a spinoff index.
