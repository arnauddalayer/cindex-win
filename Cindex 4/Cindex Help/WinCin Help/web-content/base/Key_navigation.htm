Navigation shortcuts

The following table lists the navigation shortcuts you can use when working with the main view of the index.

Keystrokes Action
DOWNARROW (with SHIFT extends any selection) Moves display down one record or entry. If any record is selected, moves the selection.
DOWNARROW (with SHIFT extends any selection) Moves display up one record or entry. If any record is selected, moves the selection.
PGDN Moves display down one page
PGUP Moves display up one page
HOME Moves display to head of index
END Moves display to foot of index
ESC Deselects records and removes highlighting
TAB Brings current selection into view
RETURN Opens the selected record (or the first of multiple selected records) for editing
CONTROL TAB (with SHIFT reverses direction) Activates next Cindex window
OTHER CHARACTERS* Treated as the leading text in a record (or a record number); Cindex displays the matching record

* When you type leading text to specify a record, Cindex displays the text you type on the status line at the bottom of the main window.
