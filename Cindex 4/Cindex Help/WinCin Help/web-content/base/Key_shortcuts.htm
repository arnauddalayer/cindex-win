Record editing shortcuts

When adding or editing records you can enter accented characters using the standard keyboard shortcuts used by Microsoft Word.

Use the following keys and keystroke combinations as shortcuts for when editing (holding down the SHIFT key together with any key shown identified with [SHIFT] selects text):

Keystrokes Action
DOWNARROW [SHIFT] Moves insertion point to next line
UPARROW [SHIFT] Moves insertion point to previous line
CONTROL DOWNARROW Moves insertion point to last line (wraps)
CONTROL UPARROW Moves insertion point to first line (wraps)
LEFTARROW [SHIFT] Moves insertion point back one character
RIGHTARROW [SHIFT] Moves insertion point forward one character
CONTROL LEFTARROW [SHIFT] Moves insertion point back one word
CONTROL RIGHTARROW [SHIFT] Moves insertion point forward one word
HOME [SHIFT] Moves insertion point to start of field (or previous field if at start)
END [SHIFT] Moves insertion point to end of field (or next field if at end)
TAB Moves insertion point to start of next field
SHIFT TAB Moves insertion point to start of previous field
CONTROL BACKSPACE Deletes text to start of word
CONTROL SHIFT BACKSPACE Deletes text to start of field containing insertion point
CONTROL DELETE Deletes text to end of word
CONTROL SHIFT DELETE Deletes text to end of field containing insertion point
CONTROL + or (keypad)+ Exchange heading with one below it (flip). Use with SHIFT to reverse preferred setting of smart flip.
CONTROL – or (keypad) – Discard heading and pull up the heading below it (half-flip). Use with SHIFT to reverse preferred setting of smart flip.
CONTROL n (N is a number key). Inserts text of heading at level N from the last-edited record
CONTROL 0 (zero) Inserts text of locator field from the last-edited record
CONTROL RETURN
or (keypad) ENTER		 Enters new/changed record in index, closes window
PGDN Enters new/changed record, moves to next
SHIFT PGDN Enters new/changed record, leaves a copy as a new record
PGUP Enters new/changed record, moves to previous
ESC Discards changes to record
CONTROL SHIFT O Restores record to its original state
CONTROL ] Increment last page number
CONTROL [ Decrement last page number
CONTROL ALT S Swap text in parentheses

To select a word, double-click it. To select a whole field, triple-click it. To extend the current selection in a unit of the size last selected, SHIFT-click.
