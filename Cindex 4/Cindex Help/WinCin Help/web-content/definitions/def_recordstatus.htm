Record Status

Each index record contains private information about the date and time it was last modified, and whether or not it is deleted, marked, labeled, or automatically generated. If you have set up Cindex to use user identifiers, the record will also contain information about the indexer who most recently modified it.

Cindex displays this information on the status line of the record window when you edit a record.

In any draft view of the index, Cindex use the following marks in the left margin to denote record status:

  • x denotes a deleted record
  • # denotes a marked record
  • · denotes a record modified during your current session with Cindex

A labeled record appears in a distinctive color (by default, red)

To find records that have particular attributes (marked, labeled, etc.) choose Find… from the Edit menu.
